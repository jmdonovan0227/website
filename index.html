Smart Brain

Smart Brain is a face-detection web application. To use the application, users must create an account from the frontend, which uses React components, hooks, and state to make Smart Brain efficient and interactive. After creating a valid account, users can paste image links into our application, which is sent to a server created using Express. The Express server sends the image link to Clarifai API to detect faces in the image. After retrieving info from Clarifai, the Express server will call a PostgreSQL database that stores user information, so it can update the number of valid image entries the user has entered into Smart Brain. And, all of the information returned from Clarifai and the database (updated image entry count) will be returned and rendered on the frontend. Smart Brain also utilizes various packages and frameworks/libraries for styling, securing user information, and more.

Maze Pathfinder

Maze Pathfinder is a maze-solving web application. Users can generate random mazes between dimensions of 5x5 up to 50x50. By default, mazes are generated with dimensions of 20x20. To solve mazes, the application has two options which are breadth-first search and depth-first search. While solving the mazes, paths will change colors to provide a visual representation of how bfs and dfs are solving the mazes. As a little extra, I also added a very quiet clicking noise while the algorithms are solving the maze and a message when the maze has been solved.
